# Tech It Up
## Description
Come and sign-up to be able to create your own blog posts and make comment on other blogs. Even you don't wish to create an account, you can still come and checkout other posts.  

## Built With
* Handlebars
* CSS
* Javascript
* Sequelize
* Node
* Express
